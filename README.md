# Financial Budget vs Actual Analysis | Power BI

## 📌 Project Overview
This project analyzes financial performance by comparing **budgeted vs actual amounts** across departments, regions, projects, and time periods. The goal is to identify **cost overruns, savings opportunities, and performance trends** using interactive Power BI dashboards.

## 🛠️ Tools & Technologies
- **Power BI** – Data modeling, DAX measures, interactive dashboards
- **Excel (CSV)** – Source dataset (50,000+ records)
- **DAX** – Budget, Actual, Variance, and KPI calculations
- **Git & GitHub** – Version control and project sharing

## 📂 Dataset
- Time Period: 2020–2025
- Size: 50,000+ records
- Key Fields:
  - Date, Department, Region, Project, Manager
  - Budget Amount, Actual Amount
  - Expense Type, Category (Revenue / Expense)

## 📊 Dashboard Features
- KPI cards for **Total Budget, Total Actual, Variance, Variance %**
- Budget vs Actual comparison by **Department**
- Monthly **Budget vs Actual trend analysis**
- Expense breakdown by **Cost Center & Expense Type**
- Interactive slicers (Department, Region, Project, Manager, Category)
- Conditional formatting for performance insights

## 📈 Key Insights
- Identified departments exceeding budget limits
- Highlighted monthly overspending trends
- Enabled drill-down analysis at project and manager level
- Improved financial transparency for decision-making

## 🚀 How to Use
1. Download the dataset from the `data` folder
2. Open the `.pbix` file in Power BI Desktop
3. Refresh data and explore the interactive dashboard
